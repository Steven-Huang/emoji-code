emoji-code
==========

kinds of code for emoji

emoji表情的名称和各种编码

##编码包括

```
unified // U+2600
docomo // U+E63E
kddi // U+E488
softbank // U+E04A
google // U+FE000
softb_unicode // \ue04a 在微信中可用到

```

##目前提供的格式有：

```
sql
json
yaml
csv
```

数据参考来源于：
[iamcal/php-emoji](https://github.com/iamcal/php-emoji)

可根据classname使用这个项目中的样式表
